# ⏱️ Specification: Timer v2 (Dynamic Time Management)

## Overview
Transform the timer from a passive number into an intuitive, dramatic indicator of dynamization flow.

## Functional Requirements
- **Progress Ring/Bar:** Add a circular or linear visual indicator that shrinks/closes as time expires.
- **Hectic Mode:** When < 10s remaining, trigger a rhythmic pulse and a red glowing "alarm" state to build group tension.
- **Cinematic "Time's Up":** Full-screen reveal (flash or glass-break effect) on expiration to ensure maximum attention.
- **Picture-in-Picture (PiP):** Allow dragging the timer to the center or minimizing it to a small icon in the toolbar without stopping the count.

## Acceptance Criteria
- [ ] Time remaining is visually obvious without reading the numbers.
- [ ] The final 10 seconds create a clear, playful sense of urgency.
- [ ] The "Time's Up" moment is impossible to miss.
