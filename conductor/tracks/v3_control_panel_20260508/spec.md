# 🎮 Specification: Control Panel & Floating Island v3.0

## Overview
Reimagine the control interface to reduce cognitive load and maximize screen space, transitioning from a fixed bar to a set of organic "Floating Islands".

## Functional Requirements
- **Floating Island Concept:** Replace the full-width `#control-panel` with a compact "pill" design that floats and expands organically.
- **Exclusive Iconography:** Design and implement a set of custom SVG icons that align with the Montserrat typography and Glassmorphism style.
- **Cascading Context Menus:** Reorganize submenus (Sounds, Links) into elegant grids or radial menus that don't push the main layout.
- **Activation Indicators:** Continuous visual feedback for active tools (e.g., pulsing border for a running Timer or active Drawing mode).
- **Smart Send Button:** Integrate the queue counter directly into a circular progress indicator within the send button.

## Acceptance Criteria
- [ ] Control panel occupies less horizontal space when not in use.
- [ ] Icons are consistent in stroke weight and style.
- [ ] Active tools have a clear, non-intrusive "running" state indicator.
- [ ] Queue badge is elegantly integrated into the main action button.
