# 📊 Specification: Polls v2 (Interactive Infographic)

## Overview
Transform the static voting bars into a lively, interactive infographic with high visual impact and physical feedback.

## Functional Requirements
- **Multicolor Gradients:** Assign a unique CSS gradient to each poll option (up to 6 distinct palettes).
- **Glass-Liquid Effect:** Add an animated "wave" or "pulsating" overlay at the top of each bar to simulate a filling liquid container.
- **Impact Micro-interactions:** Implement a "bounce" effect and a burst of particles when a vote is added or removed.
- **Winner Ceremony:** 
    - Desaturate (grayscale) losing options.
    - Intense glow effect on the winner.
    - Trigger a confetti shower using the winner's primary color.

## Acceptance Criteria
- [ ] Each bar has a distinct, vibrant color scheme.
- [ ] Adding a vote feels "tactile" through motion and particles.
- [ ] The "Check Winner" action feels like a dramatic climax.
