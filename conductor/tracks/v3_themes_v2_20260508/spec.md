# 🎨 Specification: Advanced Themes & Brand Cohesion

## Overview
Move beyond color changes by adding unique textures, ornaments, and "smart" branding that responds to the selected theme.

## Functional Requirements
- **Theme-Specific Textures:**
    - **Sketch:** Grainy paper texture and a subtle "jitter" (hand-drawn) effect on UI borders.
    - **8-bit:** CRT scanline filter and a subtle pixel grid overlay.
    - **Neon:** Simulated ground glow reflection emanating from the bottom toolbar.
- **Brand Ornaments:** Inject subtle decorative patterns into the glass panels based on the current program (e.g., neural network nodes for IA).
- **Smart SVG Logos:** Transition from static images to SVGs that can change color or apply filters to blend perfectly with theme backgrounds while maintaining accessibility.

## Acceptance Criteria
- [ ] Sketch theme feels like real paper/drawing.
- [ ] 8-bit theme has a convincing retro arcade "flicker" and texture.
- [ ] Logos are always high-contrast and integrated into the aesthetic.
