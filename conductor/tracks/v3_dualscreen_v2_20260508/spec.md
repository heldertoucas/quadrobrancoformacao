# 📺 Specification: Dual Screen & Sync v2 (Pro Studio)

## Overview
Elevate the dual-screen workflow to a "training studio" standard, giving the trainer total control over the narrative reveal.

## Functional Requirements
- **On-Air Indicator:** A pulsating virtual LED in the toolbar that confirms an active projection window is listening.
- **Digital Laser Pointer:** A cursor glow that follows the trainer's mouse position ONLY on the projector window.
- **Stage & Push Mode:** Allow the trainer to type or draw in "private" mode, revealing the update to the projector only upon hitting "Publish".
- **Safe Area Overlay:** A faint border on the control screen showing the exact boundaries of the projection aspect ratio.
- **Animated Standby:** A branded waiting screen for the projector when no content is active.

## Acceptance Criteria
- [ ] Trainer knows exactly if the projector is connected.
- [ ] "Laser" pointer helps guide students' attention.
- [ ] "Stage & Push" works as a toggleable workflow.
