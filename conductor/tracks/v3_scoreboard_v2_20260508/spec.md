# 🏆 Specification: Scoreboard v2 (Dynamic Podium)

## Overview
Move from a static score line to a dynamic ranking system that gamifies the classroom experience in real-time.

## Functional Requirements
- **Podium Layout:** Automatically reorder team cards based on current scores (top score stays central or higher).
- **Odometer Animation:** Implementation of a physical "rolling" or "pop" animation for number changes to increase satisfaction.
- **Team Identities:** Replace generic "Team A/B" with vibrant colors and thematic icons (e.g., Tech Symbols, Animals).
- **Leadership Glow:** Apply a persistent "aura" or animated gradient border to the team in the lead.
- **Mini-Mode:** Ability to collapse the scoreboard into small icons on the side to focus on text content.

## Acceptance Criteria
- [ ] Team order updates smoothly as scores change.
- [ ] Scoring a point feels like a "win" through visual feedback.
- [ ] Leadership is visually obvious at a glance.
