# 🎲 Specification: Dice & Draws v3.0 (Physical Randomness)

## Overview
Move from abstract text symbols to high-fidelity, physically-modeled random elements that build suspense and engagement.

## Functional Requirements
- **3D CSS Dice:** Construct realistic 6-sided cubes using pure CSS 3D transforms. Implement physics-based rotation (random multi-axis spins).
- **Slot Machine Draw:** Replace the instant name draw with a vertical scrolling "reel" of names that slows down (easing out) to the winner.
- **Winner Dramatization:** Apply high-impact lighting (glow) and drop shadows to the selected result.
- **Object Theming:**
    - **8-bit:** Pixelated sprites.
    - **Sketch:** Hand-drawn frames.
    - **Neon:** Glow-in-the-dark LEDs.

## Acceptance Criteria
- [ ] Dice rotation looks like a real physical throw.
- [ ] Draw suspense is maintained for at least 2 seconds via the reel effect.
- [ ] Winning result is the clear visual focal point.
